# Task React

## Installation & Running

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Libraries Used

### Core Libraries

- **React** (v19.2.0) - UI library
- **React DOM** (v19.2.0) - DOM manipulation
- **TypeScript** (v5.9.3) - Programming language
- **Vite** (v7.2.4) - Build tool and dev server

### Form Management & Validation

- **React Hook Form** (v7.66.1) - Form management
- **Zod** (v4.1.13) - Schema validation
- **@hookform/resolvers** (v5.2.2) - Zod integration with React Hook Form

### Styling & UI

- **Tailwind CSS** (v4.1.17) - CSS framework
- **clsx** (v2.1.1) - Utility for merging class names

### Routing & Notifications

- **React Router DOM** (v7.9.6) - Client-side routing
- **React Hot Toast** (v2.6.0) - Toast notifications
